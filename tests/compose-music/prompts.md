# compose-music Evaluation Prompts

Use these prompts for regression-style checks after changing the skill. The executable source of truth is `evals/compose-music.prompts.csv`.

1. Create an 8-bar dark techno loop in D dorian, Ableton-ready.
2. Turn this kick/snare/hat 16-step grid into MIDI note JSON.
3. I have a 4-bar loop. Expand it into a 64-bar arrangement.
4. Give only a bassline for this chord loop: i | VI | III | VII.
5. Implement the sketch in Ableton using ableton-cli.
6. I am stuck over-editing a loop. Give me a finish plan.
7. Write a garage drum and bass pattern with swing notes, but do not include a full song plan.
8. Create a melodic techno sketch with a composition_spec and validate-ready note JSON.
9. Diagnose why my kick and bass feel crowded in a 124 BPM house loop.
10. Make a 32-bar ambient arrangement with density and transition events.
11. Only give me a melody motif for a D dorian techno lead.
12. Make a 7/8 techno loop for Ableton but keep it implementation-safe.
13. Create a garage drum and bass hybrid loop with swing and a clear drop entry.
14. Give a synthwave chord progression only, with roman numerals and voicing notes.
15. Make an Ableton handoff plan but I did not provide a genre.
16. Repair this broken spec: handoff is missing and one browser query is path-like.
17. Make a beat-light ambient texture plan with no drum kit.
18. Write a shuffled UK garage drum grid and preserve timing metadata.
19. Create a polymeter IDM percussion idea with reset metadata but keep Ableton handoff safe.
20. Make an acid techno loop where sound design is described by acoustic intent, not preset names.
21. Expand a 16-bar minimal techno loop into a 96-bar arrangement.
22. Give only borrowed chord options for A minor, including slash or inversion examples.
23. Create a bassline and state whether it avoids, doubles, answers, or intentionally overlaps the kick.
24. Convert a validated melodic techno composition_spec into an ableton_handoff_plan artifact.
25. Make a 128-bar progressive house arrangement with density, identity carriers, and transitions.
