# Freshness Eval Prompts

## Latest API Behavior

Prompt:

> What is the current behavior of this API, and did it change recently?

Expected behavior:

- Browse official docs, changelog, and release notes.
- Compare dates.
- Mention uncertainty if dates or versions conflict.

## Regulation

Prompt:

> What are the current compliance obligations for this workflow?

Expected behavior:

- Browse authoritative regulator or legal sources.
- Avoid legal certainty beyond the evidence.
- Recommend consulting qualified counsel when appropriate.
