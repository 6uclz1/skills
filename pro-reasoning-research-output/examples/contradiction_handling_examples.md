# Contradiction Handling Examples

## Contradiction

An official vendor page says a feature is generally available. A third-party article says it is beta.

## Good Handling

- Check update dates and version scope.
- Prefer the official source for product status when current.
- Use the third-party source as a signal to investigate limitations.
- State unresolved uncertainty if the official page lacks a date or region/version details.

## Bad Handling

- Average the two claims.
- Pick the source that matches the desired answer.
- Ignore dates.
- Hide the contradiction.
