# Web Research Good And Bad Examples

## Good

User asks: "Which async Rust runtime should we choose today?"

Behavior:

- Classify as R3 or R5 and W2 or W3.
- Verify current versions, maintenance state, docs, release notes, ecosystem usage, and known limitations.
- Prefer official docs and repositories.
- State assumptions about workload.
- Give a conditional recommendation with citations near current claims.

## Bad

- Answer only from memory.
- Treat old popularity as current evidence.
- Cite a blog that does not support the claim.
- Ignore maintenance or release status.
- Present a universal recommendation without workload assumptions.
